FIELD MANUAL · 修炼手册

把「AI 帮我写的项目」
变成「我能讲清每一层」

这套教材面向有前端/移动端功底、正在切入 AI 应用开发的工程师：React、TS、Android、Java、SQL 就是你的地基，每个新概念都从你已经会的东西出发——用 TypeScript 学 Python，用 Express 学 FastAPI，用 SQL 直觉学向量检索。全部素材取自 MyOpenWeb 仓库的真实代码，学完即能讲清这个项目的每一层实现。

怎么用这套教材（建议 7–10 天过完）

  1. 顺序读：第 1–3½ 章打地基（2–3 天）→ 第 4–6 章啃核心（3 天）→ 第 7–10 章补工程与面试（2–3 天）。
  2. 对照读：每章「读代码」栏目引用的都是仓库真实文件，左边开教材、右边开 Cursor 对照源码，改一行跑一下。
  3. 动手跑：所有「动手实验」的命令都能直接复制执行（WSL venv 环境），跑通一个实验比读三遍文字有用。
  4. 面试卡自测：每章末尾的折叠问答先自己答再展开对照；第 10 章有全项目 50+ 题总集与话术模板。
  5. 主题切换：左下角可切深色模式；手机上也能读（侧栏变抽屉）。

学习心法

你不是零基础转行——你是带着多年工程直觉补一块新领域的人。教材里所有「前端类比」框都是捷径：见到 async 想 Promise、见到装饰器想注解、见到向量检索想 ORDER BY。面试官要的不是背概念，而是「这个人能把新东西挂在扎实的工程功底上」。
