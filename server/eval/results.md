# RAG 检索质量评测报告

- 时间：2026-07-04 12:30:25
- 文档：`examples/ops-manual.md`；数据集：`server/eval/dataset.jsonl`（30 条 QA）
- embedding 模型：`bge-m3`；overlap：100
- 评测方式：每条 QA 检索 top8，命中判定为片段包含期望关键串（Hit@K），MRR 取首个命中片段排名倒数的平均值；耗时含查询向量化，为端到端检索延迟。

严格人工标注集可在数据规模上来后替换 `is_relevant` 判定。
